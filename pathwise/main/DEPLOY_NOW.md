## 🆘 Troubleshooting

**Blank page after deploy?**
- Check browser console (F12) for errors
- Verify Firebase credentials are correct
- Check hosting console for build logs

**Build fails?**
- Ensure all dependencies installed locally first
- Check Node version: `node --version` (need 16+)
- Try clearing cache: `expo export --clear`

**Not loading Firebase data?**
- Verify `.env` variables in hosting dashboard
- Check Firestore security rules
- Ensure database rules allow public read (demo) or auth rules set
